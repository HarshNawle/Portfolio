## üìù Image Guidelines

1. **File Names**: Use descriptive, lowercase names with hyphens
2. **Formats**: 
   - Use `.webp` for photos (better compression)
   - Use `.svg` for icons and logos (scalable)
   - Use `.jpg` for fallback images
3. **Sizing**: Optimize for web use
4. **Alt Text**: Provide descriptive alt text in components

## üöÄ Adding New Images

1. Choose appropriate subdirectory
2. Optimize image for web
3. Use descriptive filename
4. Update components that reference the image
